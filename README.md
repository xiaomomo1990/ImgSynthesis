# ImgSynthesis

图片合成插件

### Install

* base

your html

```html
<script src="ImgSynthesis.js"></script>

```

* npm

```bash
npm i picture-synthesis --save
```

your app.js

```javascript

import pictureSynthesis from 'picture-synthesis'

```

### Usage

```javascript
const ic01 = new ImgCombination({
	mainImg:{...}
	elesImg:[...],
	type:'...'
});

```

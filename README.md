# PictureSynthesis

a lightweight javacript library for synthesizing pictures

### Install

* base

your html

```html
<script src="PictureSynthesis.js"></script>

```

* npm

```bash
npm i picture-synthesis --save
```

your app.js

```javascript

import PictureSynthesis from 'picture-synthesis'

```

### Usage

```javascript
const ic01 = new PictureSynthesis({
	mainImg:{...}
	elesImg:[...],
	type:'...'
});

```
